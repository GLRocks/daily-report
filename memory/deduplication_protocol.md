# 重复内容控制机制 v1.0（2026-06-04生效）

## 问题诊断
1. **S2投资人论点**：旧quote复用（1-5个月前的言论重复出现）
2. **S4模型表格**：静态模板化，参数/版本不随日更新
3. **跨日Topic重复**：去重日志执行不到位

## 强制规则

### 规则1：投资人论点时效门槛（S2）
- **只收录近7日内的新言论**（发言日期距离今日 ≤7天）
- 超过7日的旧quote → **禁止收录**，除非该人物当日有明确的新表态
- 人物覆盖范围扩展：不仅限于Andreessen/Dalio/Fink/Huang，新增：
  - 当日出现在Reuters/Bloomberg/FT的CEO/CFO/CTO言论
  - 当日财报电话会上的关键表态
  - 当日国会/监管听证会证词
- 每条quote必须标注：**发言日期 + 原始来源URL**，无日期/来源 → 不收录

### 规则2：模型表格动态化（S4）
- **禁止静态模板**：表格中任何参数（context窗口、benchmark分数、定价、融资额）必须当日验证
- 如果某公司当日无新信号 → **该行标灰/标注"暂无新信号"**，禁止复制昨日数据填充
- 新增字段："当日更新"列，标注哪些参数是今日新确认、哪些是历史背景
- 7家公司中，当日无新动态的公司 → 只保留ticker+名称，其余留空

### 规则3：去重日志强制执行
- **搜索前必须读取** `/root/.openclaw/workspace/memory/daily_topics_log.md`
- **近3日已覆盖的Topic** → 禁止收录，即使当日有"新进展"，仅当进展具有独立投资含义时才跟进
- **近4-7日** → 仅收录突破性进展（非渐进式更新）
- 每日生成后追加日志，格式：`YYYY-MM-DD | 板块 | 核心Topic | 来源 | 投资含义摘要`

### 规则4：股票面板文字差异化
- 推荐文字（`.stock-reason`）必须反映**当日具体催化剂**，禁止复制模板
- 如果当日无新催化剂 → 写"暂无新催化剂，维持原评级"，禁止复制昨日文字
- 价格/涨跌幅每日更新，但文字必须差异化

### 规则5："新信号"标记系统
- 每条信息必须自评："当日新信号" vs "背景补充"
- 每个板块中，"背景补充"类信息不得超过30%
- 如果某板块当日无新信号 → 允许整块标灰/标注"当日无重大投资信号"

## 问责机制
- 自检清单增加第9项：「近3日日志中无重复Topic？」
- 自检清单增加第10项：「S2中所有quote日期距离今日≤7天？」
- 自检清单增加第11项：「S4表格中所有参数已当日验证？」
- 任一检查不通过 → 当日对应板块标灰/留空，宁可信息少也不重复
