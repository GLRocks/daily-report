## 股票价格显示问题修复

### 问题
用户多次反馈股票推荐卡片中价格未显示。经排查，发现以下问题链：

1. **初始版本(v5/v6)**：价格行使用 `display:flex` + CSS变量(`var(--border)`等)，Chrome headless生成PDF时价格行可能渲染异常
2. **验证方法错误**：最初用 `grep` 搜索PDF二进制文件，发现0个价格字符串 → 误以为PDF中没有价格
3. **真相**：安装 `pdftotext` (poppler-utils) 后验证，PDF v7 确实包含价格数据（MSFT $415.12、GOOGL $400.80、AMD $455.19、NVDA $215.20 等）
4. **网页版本正常**：用户截图显示网页中价格已正确显示

### 修复措施
1. CSS从 `flex/gap/CSS变量` 改为 `block/inline-block/硬编码颜色值`
2. 补全已知价格：GOOGL $400.80 (+0.71%)、AMD $455.19 (+11.44%)
3. 推送 v7 PDF + 新 Cloudflare tunnel 链接

### 教训
- PDF文本提取不能用 `grep` 二进制文件，必须用 `pdftotext`
- Chrome headless `--print-to-pdf` 的CSS兼容性：避免复杂flex布局，使用传统block/inline-block更可靠
- 用户可能在不同环境查看（网页/PDF/Kimi预览），需要分别验证
